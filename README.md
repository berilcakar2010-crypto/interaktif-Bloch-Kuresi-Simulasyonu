# interaktif-Bloch-Kuresi-Simulasyonu
Bloch küresi, kuantum mekaniğinde iki seviyeli bir sistemin (kübit) saf durum uzayını görselleştirmek için kullanılan 3 boyutlu birim küre geometrik modelidir. Felix Bloch'un adını taşıyan bu küre, kübitin süperpozisyon ve faz bilgilerini küre yüzeyindeki noktalarla (kutup ve ekvator) temsil eden temel bir kuantum bilişim aracıdır.
